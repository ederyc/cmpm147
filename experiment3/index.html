Experiment X - Procedural Tile World

Description

This experiment generates a procedurally created world using tiles, Perlin noise, and seeded randomness. The user can toggle between an overworld and a dungeon style.

Technical

The project uses a seeded RNG to build a grid of characters and render them using a tileset. A button toggles tile logic between a natural overworld and a structured dungeon layout.

Reflection

This setup helped explore tile-based procedural generation with minimal code and maximum flexibility.
